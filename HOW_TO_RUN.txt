To run the project:

1. Clone the Repository
    in terminal enter the command git clone https://github.com/AICourseworkGroup/Spider.git
    then change directory by typing "cd Spider".

2. Installing dependencies 
    The requirements.txt has all the required dependencies, needed to run the application.
    pip install -r requirements.txt.

3. Run the Genetic Algorithim and Neural Network
    To run the Genetic Algorithim and Neural Network, you must type "python main" in the terminal.
